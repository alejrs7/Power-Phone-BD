#  Power-Phone – Sistema de Gestión de Base de Datos

##  Descripción
**Power-Phone** es una base de datos relacional para la gestión de una tienda de accesorios móviles.

## Estructura del repositorio
- `/Diagramas` → Modelos gráficos (ER y Relacional)
- `/ScriptsSQL` → DDL.sql, DML.sql, Consultas.sql

##  Tecnologías
MySQL 8.4, MySQL Workbench, Draw.io, Git
